# TradeMinds — AI Autonomous Trading System

Fully autonomous AI-powered trading bot supporting Forex, Crypto, Commodities, Stocks and Indices.

## Tech Stack
- **Backend:** Python + FastAPI
- **Frontend:** React + TypeScript
- **Database:** PostgreSQL + Redis
- **AI Engine:** Claude API (Anthropic)
- **Brokers:** CCXT (Crypto) + MetaAPI (MT4/MT5) + IBKR
- **Calendar:** MyFXBook XML Feed

## Features
- 7/24 autonomous trading across all markets
- 6-layer risk management system
- Backtest engine with equity curve
- Economic calendar integration
- Real-time WebSocket dashboard
- Telegram + Email notifications
- PDF/Excel report generation
- Manual position close (only user intervention)

See `deploy/setup.sh` for one-command VPS setup.
